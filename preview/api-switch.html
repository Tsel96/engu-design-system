Switch

Yellow on / muted off. Use for binary preferences that take effect immediately. For settings that require a Save step, use a Checkbox.

API

PropTypeDefaultDescription
checkedbooleanundefinedControlled state.
defaultCheckedbooleanfalseUncontrolled initial state.
onChange(checked: boolean) => voidundefinedCalled when the value changes.
size"sm" | "md""md"sm 30×15 · md 51×27. Both have the same knob-stretch press affordance.
disabledbooleanfalseLocks the track to yellow-50 with a 10% overlay.
labelstringundefinedVisible label. Recommended; bare switches need aria-labelledby.
helperTextstringundefinedOptional description below the label.
aria-labelstringundefinedRequired when no visible label.
namestringundefinedFor form submission.
valuestring"on"Submitted value when checked.
