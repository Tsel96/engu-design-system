Modal

Centered dialog with an ink-green scrim. Reserve for high-friction decisions (cancel, delete, confirm). For non-blocking, use BottomSheet on mobile and Popover on desktop.

API

PropTypeDefaultDescription
openrequiredbooleanundefinedControlled visibility.
onCloserequired() => voidundefinedCalled on backdrop click, Esc, or close button.
titleReactNodeundefinedDisplay heading. Apparat sizing — keep ≤4 words.
descriptionReactNodeundefinedBody copy below the title.
childrenReactNodeundefinedCustom body (overrides description).
size"sm" | "md" | "lg""md"sm 320 · md 440 · lg 600.
intent"neutral" | "danger""neutral"Danger reframes the primary action as red.
primaryAction{ label, onClick, loading? }undefinedPrimary CTA. Pass loading for async confirmations.
secondaryAction{ label, onClick }undefinedOptional secondary button.
hideClosebooleanfalseHide the top-right close X.
hideBackdropbooleanfalseSkip the ink scrim. Reserve for in-app coachmarks.
preventBackdropClosebooleanfalseBackdrop click no longer closes.
initialFocusRefObjectprimary actionElement to focus on open. Do not auto-focus a destructive button.
aria-labelledbystringautoWired automatically to title.
<Modal
  open={confirming}
  onClose={() => setConfirming(false)}
  title="Cancel subscription?"
  description="Your access continues until 12 June."
  intent="danger"
  secondaryAction={{ label: "Keep plan", onClick: dismiss }}
  primaryAction={{ label: "Cancel", onClick: cancel, loading: pending }}
/>
