Button

Pill-shaped action. Three emphasis levels (primary, secondary, tertiary) × four sizes (xs/sm/md/lg). Renders as <button> or <a> depending on href.

API

PropTypeDefaultDescription
childrenrequiredReactNodeundefinedButton label.
emphasis"primary" | "secondary" | "tertiary""primary"primary = yellow fill, button-stack shadow. secondary = white fill, 1px border. tertiary = transparent, hover-bg only.
size"xs" | "sm" | "md" | "lg""md"xs 36 · sm 44 · md 54 · lg 64.
iconLeftReactNodeundefinedLeading icon.
iconRightReactNodeundefinedTrailing icon (chevron, external-link).
iconOnlybooleanfalseSwitches to icon-button geometry (square or circle).
shape"pill" | "square" | "circle""pill"Pill for labeled. Square/circle for icon-only.
loadingbooleanfalseReplaces iconLeft with a spinner. Disables clicks. Width is preserved.
disabledbooleanfalsePointer-events off, 40% opacity.
intent"neutral" | "danger""neutral"danger recolors primary to red.
hrefstringundefinedWhen set, renders as <a>.
onClick(e) => voidundefinedClick handler. No-op while loading or disabled.
type"button" | "submit" | "reset""button"Default is button — never submit unless inside a form.
fullWidthbooleanfalseStretches to container width.
aria-labelstringundefinedRequired when iconOnly is true.

Usage

<Button emphasis="primary" size="lg" onClick={start}>
  Start an adventure
</Button>

<Button emphasis="primary" loading>
  Saving…
</Button>
