Card

Surface primitive — a 20px-radius container with a soft shadow on cream/canvas. Pass surface to switch tone, onClick to make it interactive.

API

PropTypeDefaultDescription
childrenrequiredReactNodeundefinedCard content.
surface"default" | "tinted" | "brand" | "inverse""default"default = white, shadow-card. tinted = cream, no shadow. brand = engu-green, white text. inverse = ink-green, white text.
radius"sm" | "md" | "lg" | "xl""lg"sm 12 · md 16 · lg 20 · xl 24.
padding"none" | "sm" | "md" | "lg""md"none 0 · sm 16 · md 24 · lg 32.
elevation"flat" | "raised" | "lifted""raised"flat = no shadow. raised = shadow-card. lifted = shadow-lift.
onClick(e) => voidundefinedWhen set, renders as a button and adds hover tint.
hrefstringundefinedRenders as <a>. Mutually exclusive with onClick.
disabledbooleanfalseOnly meaningful when onClick/href set.
asElementType"div"Render as a different element.

Composition

Card has no header / body / footer subcomponents — compose with Stack, Row and headings directly. Keeps the abstraction thin.

<Card surface="brand" padding="lg" onClick={open}>
  <Stack gap={3}>
    <h3>Forest puzzle</h3>
    <p>Find the missing seeds.</p>
  </Stack>
</Card>
